hexo 几个常用命令 :

  • 1 hexo g (hexo generate) 用于生成静态文件
  • 2 hexo s (hexo server) 启动web服务
  • 3 hexo d (hexo deply) 部署到远端(比如github)
  • 4 hexo new [layout] “postName” 新建文章

  • 5 hexo new page “pageName” 新建页面
